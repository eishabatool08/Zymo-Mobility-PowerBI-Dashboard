# 🚖 Zymo Mobility - Ride-Hailing Power BI Dashboard

An interactive, dual-theme (Light & Dark Mode) Power BI dashboard analyzed for **Zymo Mobility** (formerly Uber data insights). This project showcases key performance indicators (KPIs) regarding revenue, booking patterns, vehicle types, and customer satisfaction.

## 📈 Key Metrics & Features Captured
* **Total Revenue:** $52M generated across all booking segments.
* **Completed Bookings:** 93K successful rides tracked.
* **Distance Traveled:** 2.51M total distance with an average of 24.64 per ride.
* **Status Breakdown:** Visual tracking of Completed, Cancelled, and Incomplete rides via donut charts.
* **Trends Analysis:** Monthly time-series analysis for booking values and completed ride distribution.
* **Customer & Driver Feedback:** Rating score matrix filtered by vehicle types (Auto, Prime, Mini, etc.).
